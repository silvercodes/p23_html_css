О гитарах

Гитара — струнный щипковый музыкальный инструмент, один из самых распространённых в мире.

Типы гитар

  • Акустическая гитара

    Производит звук благодаря резонансу корпуса.

  • Электрическая гитара

    Использует электромагнитные звукосниматели для преобразования колебаний струн в электрический сигнал.

  • Классическая гитара

    Традиционный вариант с нейлоновыми струнами, используется преимущественно в классической музыке.

Аксессуары для гитар

Для игры на гитаре вам понадобятся медиаторы, ремни, тюнеры и другие аксессуары.
